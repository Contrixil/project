# 휠체어에서 살아남기 - 1인칭 3D 시뮬레이션

브라우저에서 바로 실행되는 1인칭 3D 시뮬레이션 게임입니다.  
플레이어는 WASD로 이동하고, 마우스로 시점을 조작하며, 특수 바퀴와 특수 슈트를 사용해 제한 구간을 통과한 뒤 버스에 탑승하는 것이 목표입니다.

## 게임 목표

- 버스 정류장으로 이동
- `narrowPath` 구간은 **특수 슈트**를 활성화해야 통과 가능
- `steepPath` 구간은 **특수 바퀴**를 활성화해야 통과 가능
- 버스 문 앞까지 도달하면 탑승 성공

## 조작 방법

- `W` : 전진
- `A` : 왼쪽 이동
- `S` : 후진
- `D` : 오른쪽 이동
- 마우스 : 시점 조작
- `E` : 특수 슈트 활성화 / 해제
- `F` : 특수 바퀴 활성화 / 해제

## 실행 방법

1. `index.html` 파일을 저장합니다.
2. 브라우저에서 `index.html`을 엽니다.
3. 바로 플레이할 수 있습니다.

## 특징

- 1인칭 시점 기반 이동
- 3D 형태의 간단한 월드 표현
- 특수 슈트와 특수 바퀴의 구간별 역할 분리
- `isSpecialSuitActivated`, `isSpecialWheelActivated` 상태값으로 장치 전환 관리
- 특수 바퀴 활성화 시 시각 효과 표시
- 회색 바닥과 정리된 주변 그래픽
